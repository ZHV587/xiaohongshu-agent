# 小红书文案智能体 WebUI 重构与高保真微交互设计说明书 (Spec)

日期: 2026-06-18
状态: 已批准 (Approved)
作者: Antigravity

---

## 1. 核心视觉与交互设计目标
重构现有的 PC 桌面端分屏工作台（Split-Screen Workbench），提升整体设计的艺术感与品牌调性，并引入极具品质感的微交互动效（Micro-interactions）。本 Spec 覆盖登录、侧栏历史、工作区控制、手机实时预览、原位编辑器、飞书协作闭环以及命令面板七大模块。

---

## 2. 全局视觉设计系统 (Oats/Coral/Charcoal Design Tokens)
* **主色调代币**：
  * `oats` (温馨燕麦色)：`#FAF6F0`。作为 App 主背景底色，营造“手账纸张”般的舒适书写感。
  * `coral` (小红书珊瑚红)：`#FF2442`。作为高亮、点缀色及主行动按钮。
  * `charcoal` (深木炭黑)：`#2A2A2A`。用于文案正文，提供高易读性。
  * `topicblue` (话题蓝)：`#007FFF`。用于 Hashtag 标签及同步态。
* **物理材质与空间层级 (Surfaces & Depth)**：
  * **Level 0 (底座背景)**：Oats 底色配合微妙的珊瑚红到燕麦色的**径向漫反射光晕**。
  * **Level 1 (卡片/面板)**：纯白色卡片，配备 `0.5px` 极细的半透明描边（`border-coral-light`）及软阴影。
  * **Level 2 (浮窗与气泡)**：采用 `backdrop-blur-md bg-white/75 border border-white/50 shadow-2xl`，营造通透温润的**毛玻璃 (Glassmorphism)** 质感。

---

## 3. 各模块重构与微交互规范

### 3.1 登录门禁 (AuthGate)
* **氛围背景**：以中央登录卡片为原点，向外放射出极柔和的 `#FFEDF0`（Coral-light）到 `#FAF6F0`（Oats）的径向渐变背景。
* **卡片材质**：使用 `backdrop-blur-md bg-white/80 border border-white/50 shadow-2xl` 磨砂玻璃质感。
* **3D 卡片翻转**：切换“扫码模式”时，卡片以中轴线为轴进行 180 度 Y 轴立体翻转，采用 Framer Motion 的 `spring` 弹性物理动效（`stiffness: 120, damping: 14`），带点轻微的摇晃惯性。
* **成功跳跃**：扫码成功后，勾号图标从小变大弹跳（Pop animation），并伴随极轻微的卡片震动回弹。

### 3.2 历史会话侧边栏 (ThreadHistory)
* **活动项条**：当前选中的对话卡片背景使用 `bg-oats-dark`，左侧配备 `3px` 宽的 Coral 红色圆角边框指示条。
* **列表平滑退场**：使用 `AnimatePresence` 对历史卡片的删除进行高度与透明度动画过渡，卡片渐隐同时高度缩卷至 `0px`，下方的列表流畅地向上浮动填补，避免硬性位移。
* **删除平滑滑出**：鼠标悬停在卡片上时，卡片右侧滑动展出垃圾桶图标，悬停状态下图标会产生微弱的抖动动画。
* **按压反馈**：“新建对话”等动作按钮添加 `-5%` 的物理按压缩放（Scale down）反馈。

### 3.3 分屏控制与 Tab 切换 (Workspace Splits & iOS-style Segmented Tabs)
* **双视窗过渡**：分屏工作台在开启/收起侧边栏时，左侧聊天面板与右侧预览面板采用平滑的弹性滑动缩放（Spring width transform），比例自动适配。
* **Tab 切换滑块游走**：“手机预览”与“飞书协作”Tab 切换采用 **macOS Segmented Slider** 风格：外框为圆角药丸框，活动项背景为一个白色的悬浮微阴影卡片，当切换 Tab 时，该阴影卡片会在两个按钮下方横向平滑“滑行游走”（使用 `layoutId` 实现）。

### 3.4 手机预览模拟器 (Live Phone Preview)
* **外壳细节**：高保真 iPhone 外壳，配备 `8px` 黑色精细圆角壳体、磨砂质感灵动岛刘海，带亮色极细描边。
* **多图 Carousel**：图片切换时，图片伴随水平滑入滑出（Slide）与淡入淡出（Fade）的无缝缓动过渡。
* **底部互动栏可点亮**：点赞（心形）与收藏（星形）可点选交互：
  * 点击时，图标自动点亮填充为 Coral 红色，且心形/星形产生一次 Q 弹的缩放震荡动画（`scale: [1, 1.45, 0.9, 1.1, 1]`），同时上方跳出一个淡入淡出的 `+1` 数值浮动动效。
* **瀑布流卡片视窗**：切换至“瀑布流”模式时，显示高保真的双列小红书笔记卡片，包含 3:4 封面图、双行截断标题（12~14字限制并加省略号）及博主头像标识。

### 3.5 原位编辑器 (Inline Editor)
* **激活拉升**：点击手机中文本时，文本块淡出，原位编辑器面板从下方平滑向上推入。
* **自适应高度 (Auto-expanding Textarea)**：文案输入框去除垂直滚动条，高度完全随着文本换行自动无缝长高。
* **圆形字符指示器 (SVG Circular Progress)**：字数输入框角落显示一个 SVG 圆环进度条。随着字数增加，圆环顺时针填满。颜色根据数值自动演变（绿 -> 橙 -> 珊瑚红）。当字数超出 1000 字官方限制时，圆环变红并触发编辑卡片**微弱的左右抖动（Shake animation）**，警示排版受限。
* **Emoji / Tag 果冻胶囊**：表情与标签 pill 按下时产生 `-8%` 按压微缩，放开时回弹并插入光标位置。
* **AI 正在生成流 (Streaming Typewriter)**：当 AI 正在向预览器打字流式更新文案时，字尾配备一个**闪烁的珊瑚红打字光标 (Blinking Cursor)**，伴随弱粉色的文字模糊背景脉冲，视觉上更具生命力。

### 3.6 飞书协作控制台 (Feishu Integration Console)
* **同步抛物线动效**：点击同步到飞书多维表格时，飘出一个带淡淡粉色拖尾淡出（Tail-fade）的 “🍠 正在同步...” 抛物线飞行图标，从左侧发送区划过弧线飞入右侧 Feishu Tab。
* **同步校验流水日志**：进度展现为四步带有脉冲 Loading 状态的动态进度条，下方同时展示一条滚动更新的“终端风命令行同步日志”以体现专业协同质感。
* **飞书多维表格直达**：同步成功后，面板上平滑浮现出一个发光的 `🔗 打开飞书多维表格` 按钮，点击直接新开标签页定位至飞书表格，形成完美闭环。
* **飞书卡片效果预览**：提供一个小型面板展示 Bot 发送至飞书群聊的卡片渲染形态。

### 3.7 Ctrl+P 指令中心 (Raycast-style Command Palette)
* **设计风格**：置顶展示的磨砂半透明搜索框，配备极细的白色高光边框和四周大范围的暗化模糊。
* **极客键盘控制**：
  * 支持键盘 `↑ / ↓` 键在指令（如 `/polish`, `/shorten`, `/tags`）列表中上下平滑移动高亮背景（Oats 高亮卡片）。
  * 敲击 `Enter` 回车键直接执行当前高亮指令；敲击 `Esc` 键退出，实现纯键盘的极速文案处理。

## 4. 验证计划

1. **视觉保真度对比**：在 Chrome DevTools 比较 React 最终产出与 `mockup.html` 的每个阴影、描边 and 配色代币，确保一致性。
2. **动效流畅度测试**：在大容量文案或快速切换 Tab 时，帧率保持 60fps，不出现卡顿或布局闪烁（Layout Shifts）。
3. **键盘热键可用性**：连续敲击 `Ctrl+P` -> `↓` -> `Enter`，确认流畅调度 AI 瘦身等任务。
4. **飞书同步链路测试**：执行同步，观察抛物线动画、步进指示与飞书表格直达链接的生成。
