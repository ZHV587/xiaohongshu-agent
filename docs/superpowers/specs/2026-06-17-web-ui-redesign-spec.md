## 2. 界面设计规范与组件层级

### 2.1 配色方案 (Color Token Architecture)
* **`oats` (温馨燕麦色)**：`#FAF6F0`。主画布背景色。为创作者提供类似实体“纸质手账”的温润写字体验，避免刺眼白屏。
* **`coral` (珊瑚红)**：`#FF2442`。点缀色与主行动按钮，继承小红书官方品牌调性。
* **`charcoal` (深木炭黑)**：`#2A2A2A`。文案正文排版，提供温和的高易读性。
* **`topicblue` (话题蓝)**：`#007FFF`。Hashtag 标签与同步状态高亮。

### 2.2 空间层级与分隔 (Surface & Depth)
* **Level 0 (App 背景)**：使用 `#FAF6F0` (oats) 作为主底色。
* **Level 1 (面板/侧栏/工作区)**：采用 `#FFFFFF` (纯白)，边缘加 `border-[0.5px] border-coral-light`（极细的珊瑚红调和线）以及向内虚化投影。
* **Level 2 (浮窗与气泡)**：使用 `backdrop-blur bg-white/90` 营造精致的毛玻璃效果。

## 3. 核心功能组件详述

### 3.1 顶部状态栏与增量授权
* **飞书 CLI 状态显示**：实时拉取后台 `lark-cli` 状态，绿色闪烁点表示应用（bot）就绪。
* **登录态过期重连**：当检测到个人（User）授权过期时，状态栏出现警示。右下角同步面板会静默浮现**飞书登录二维码**（使用 `lark-cli auth qrcode` 生成的 QR envelope），引导用户扫码增量开通缺失的 scopes 权限，体验自然流畅。

### 3.2 爆款选题卡片组件 (Topic Selector Card)
* 选题展示从冷冰冰的“文本列表”转为**微拟物边框的互动卡片**。
* 每张选题卡片带有一个明显的 🔥 爆款热度标签（如 `🔥 爆款率 96%`）。
* 点击卡片会触发左侧聊天流提交交互，并激活右侧手机预览的流式打字渲染。

### 3.3 手机端笔记预览框 (Mobile Live Preview)
* 在右侧画布固定一个 iPhone 形状的预览框，实现正文和图片的“所见即所得”：
  - **图片封面区**：支持从本地拖拽图片完成快速合成与封面渲染。
  - **正文打字态**：智能体输出时，右侧手机屏幕的文案实时流动刷新，并将 Emoji 和 `#Hashtags` 自动高亮。

### 3.4 飞书深度同步控制台 (Feishu Integration Console)
在右侧分栏中提供第二个 Tab，专用于团队协同：
* **多维表格同步控制**：明确展示当前绑定的是飞书哪一张表、哪一行（如 `第 4 行`），点击一键写入。
* **飞书群聊推送**：支持下拉选择企业内的“文案审核群”（动态读取飞书群 ID），点击直接由 Bot 向群聊发送一张精美的审核通知卡片。
