# 用户使用场景与偏好

## 语言与场景
- **用户基本在中文场景下使用本系统**:面向小红书中文创作者,查询、选题、文案、检索语料绝大多数为中文。
- 涉及检索/嵌入/排序/提示词等设计与调参时,**默认以中文为主场景**:
  - 嵌入查询指令前缀使用中文模板(`_DEFAULT_QUERY_INSTRUCTION`)。
  - 相关度阈值、召回质量等经验值以中文查询在生产语料上的实测为准(已验证 `XHS_EMBEDDING_RELEVANCE_FLOOR` 默认 0.50:中文模板下相关查询 ≥0.58、无关 ≤0.48)。
  - 评测、标定、示例数据优先用中文,不要用英文样本代表真实分布。
- 面向用户的回复用中文。

## 交流偏好
- 直接、简洁;先给结论与依据,再给细节。
- 改动前讲清根因与风险;遵循项目铁律(根本性修复、不打补丁、相关测试一并重写)。
